Typography Rules (New in V2)
  • No text-4xl on labels — max 2rem for inline values
  • Bebas Neue for page titles only, not section headers within pages
  • Line height 1.4 for lists, 1.6 for prose
  • Colour communicates state — don't also add icons for state
  • Numbers right-aligned, always
  • Uppercase labels ≤ 0.7rem only
